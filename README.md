# Personal Economic Guardian (PEG)

## Overview
PEG is an agentic AI system designed to protect users from digital fraud and optimize economic decisions.

## Key Feature: Forensic Guardian
- Hybrid scam detection (ML + Behavioral + LLM)
- Real-time decision engine
- Honeypot agent for active scam engagement

## Tech Stack
- Python
- FastAPI (planned)
- Groq (LLM)
- Scikit-learn (ML)
- LangGraph (planned)

## Architecture
Detection → Reasoning → Decision → Action
